전기 에너지 데이터

건물에너지관리시스템을 통해 수집된 전기의 사용량 정보를 제공합니다. 단, 단독주택과 500세대 미만 공동주택은 대상에서 제외합니다.

출처: 건축데이터 개방 건축데이터 민간개방시스템
요약: 건축데이터 민간개방시스템에서 제공한 전기 에너지 소비 데이터. 각 건물의 전기 사용량을 포함합니다.
사이트: https://open.eais.go.kr/
사용범위: 2023년 1월부터 12월까지
